# nextgensay

nextgensay uses `cowsay` and `fortune` to print TNG quotes on the command line.

```
 _________________________________________ 
/ If there's nothing wrong with           \
| me...maybe there's something wrong with |
| the universe!                           |
|                                         |
\ -- Beverly Crusher, "Remember Me"       /
 ----------------------------------------- 
    \
     \
      \
                _____.-----._____
   ___----~~~~~~. ... ..... ... .~~~~~~----___
=================================================
   ~~~-----......._____________.......-----~~~
    (____)             |   /          (____)
      ||           _/   |   _           ||
       \_______--~  //~~~\  ~--_______//
        `~~~~---__   \___//   __---~~~~'
                  ~~-_______-~~
```
